🎨 Tutorial: Background Dinamis Berdasarkan Jam

📅 Jam saat ini: --:-- | Tema: --

STEP 1: Ambil Jam dari Device User

Gunakan JavaScript untuk mendapatkan jam saat ini:

const now = new Date(); // Buat object tanggal/waktu const hours = now.getHours(); // Ambil jam (0-23) const minutes = now.getMinutes(); // Ambil menit (0-59) console.log(hours); // Output: 14 (jam 2 sore) console.log(minutes); // Output: 30 (30 menit)

Penjelasan: getHours() mengembalikan angka 0-23 (format 24 jam)

STEP 2: Buat CSS Class untuk Setiap Waktu

Definisikan warna background untuk setiap periode waktu:

.night { background: linear-gradient(135deg, #0a0e27, #1a1a3e); } .morning { background: linear-gradient(135deg, #ff6b35, #ffa500); } .noon { background: linear-gradient(135deg, #87ceeb, #add8e6); }

STEP 3: Tentukan Class Berdasarkan Jam

Gunakan if-else atau switch untuk menentukan class mana yang dipakai:

function getTimeClass(hours) { if (hours >= 0 && hours < 5) { return 'night'; } else if (hours >= 5 && hours < 7) { return 'early-morning'; } else if (hours >= 7 && hours < 11) { return 'morning'; } else if (hours >= 11 && hours < 14) { return 'noon'; } else if (hours >= 14 && hours < 17) { return 'afternoon'; } else if (hours >= 17 && hours < 19) { return 'evening'; } else { return 'dusk'; } }

STEP 4: Terapkan Class ke Element

Gunakan JavaScript untuk mengubah class di element:

const timeClass = getTimeClass(hours); const body = document.querySelector('body'); // Hapus class lama body.className = ''; // Tambah class baru body.classList.add(timeClass);

STEP 5: Update Otomatis dengan setInterval

Buat background update otomatis setiap detik atau menit:

// Update setiap 1 menit (60000 milidetik) setInterval(() => { updateBackground(); }, 60000); // Atau update setiap detik setInterval(() => { updateBackground(); }, 1000);

Tips: Update setiap menit lebih efisien daripada setiap detik

💡 Penjelasan Logika Lengkap:

Contoh: Jika jam saat ini adalah 14 (2 sore)

Kode akan cek: "Apakah 14 >= 14 && 14 < 17?" → YA
Maka timeClass = 'afternoon'
Lalu element body akan mendapat class "afternoon"
Dan background akan berubah menjadi warna sore (orange + biru)
